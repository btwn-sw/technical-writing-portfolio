# Technical Writing Portfolio

<br>

## What you'll get from this

This portfolio demonstrates bilingual (English/Korean) technical writing across four different kinds of work: writing API documentation end to end, designing the information architecture for a developer portal, designing a knowledge system for a fast-moving organization, and building — and documenting — an original AI-powered tool from scratch.

Each folder is a self-contained case study. You can read them in any order, but they're listed below in a sequence that moves from direct documentation execution toward original product design.

<br>

## Structure

```
technical-writing-portfolio/
├── developer-portal/
│   └── eventbrite-api/
│       ├── docs/en/
│       │   ├── api/            (api-reference, error-reference, quick-reference)
│       │   ├── examples/       (code-examples)
│       │   ├── get-started/    (quick-start)
│       │   ├── guides/         (authentication, pagination, response_handling, troubleshooting)
│       │   ├── reference/      (glossary)
│       │   └── tutorials/      (first-api-call)
│       └── docs/ko/
├── hidoc/
├── ia-design/
└── knowledge-systems/
```
